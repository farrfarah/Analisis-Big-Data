# 📊 Sentiment Analysis of “Log-In” Podcast (YouTube)

This project analyzes audience sentiment toward the “Log-In” podcast on YouTube during Ramadan 2024. The podcast, hosted by Deddy Corbuzier, is one of the most popular digital talk shows in Indonesia.

## 📌 Objective
To classify public sentiment (positive, negative, neutral) from YouTube comments using machine learning techniques.

## 📂 Dataset
- 212,827 YouTube comments
- Collected via web scraping

## ⚙️ Methodology
- Text preprocessing (cleaning, tokenization, normalization)
- Sentiment classification using Naive Bayes
- Big data processing with Apache Spark

## 📈 Results
- Accuracy: ~95%
- ROC-AUC: ~0.78 – 0.85
- Majority sentiment: Positive

## 🛠️ Tech Stack
- Python
- Apache Spark (PySpark)
- Machine Learning (Naive Bayes)
- NLP
